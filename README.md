# Ecozed Prompt Generator

مولد أوامر التسويق والتصميم والإعلانات
جعل الإعلانات واختبار المنتجات أكثر سهولة

أداة ويب مجانية ومفتوحة المصدر لتوليد أوامر احترافية للصفحات الهبوطية والصور والنصوص الإعلانية، مبنية بـ React + Vite + Tailwind CSS.

## المميزات

### 1. صفحة هبوط (Landing Page)
توليد أوامر تصميم للصفحات الهبوطية الإعلانية الطويلة (9:32):
- **وضع متقدم** — تعبئة معلومات المنتج يدوياً (الاسم، الفوائد، السعر، الألوان، التقييمات)
- **وضع سريع** — إدخال رابط صفحة المنتج فقط، والذكاء الاصطناعي يستخرج المعلومات تلقائياً
- **6 أنماط تصميم**:
  - افتراضي — النمط الكامل المقسم
  - تخفيضات وسرعة — نبرة حادة مع عد تنازلي
  - سرد قصصي — سينمائي درامي
  - فخامة — راقي وأنيق
  - مشكلة وحل — PAS
  - مقارنة — جدول مقارنة مع المنافسين

### 2. صورة إعلانية (Image Creative)
توليد أوامر لصور إعلانية مربعة (1:1) مناسبة للسوشيال ميديا:
- **إدخال يدوي** — تعبئة بيانات المنتج كاملة
- **رابط منتج** — إدخال رابط والذكاء الاصطناعي يستخرج المعلومات
- **6 أنماط هيكلية** (تحدد عدد الأقسام وموقع CTA بينما يبني AI المشهد المناسب للمنتج):
  - افتراضي — هيكل قياسي (5 أقسام + CTA)
  - صحي — هيكل نظيف مع تركيز على النظافة والسلامة
  - فخم — هيكل فاخر مع مقارنة Before/After
  - جريء — هيكل عصري مع أيقونات ديناميكية
  - طبيعي — هيكل عضوي مع عناصر طبيعية
  - تقني — هيكل مستقبلي مع عناصر HUD
- **توليد نص الإعلان** *(اختياري، متاح فقط في الصورة الإعلانية)* — إنشاء نص تسويقي متوازن (Hook → Pain → Solution → CTA) مع الصورة

### 3. فيديو إعلاني (Video Creative)
قريباً — توليد أوامر فيديوهات إعلانية احترافية.

### خيارات مشتركة
- **اختبار A/B** — توليد أمرين بتوجهين تسويقيين مختلفين
- **لا تولد بشر** — منع ظهور أي أشخاص في التصميم
- **رجال فقط** — توليد رجال فقط دون نساء
- **فرض اللغة** — التأكد من أن كل النصوص بلغة الجمهور المستهدف
- **فحص اللغة النهائي** — تذكير غير شرطي في جميع الأوامر باللغة المطلوبة
- واجهة عربية بالكامل مع دعم RTL
